# SlideLeaf AI Playbook Sources

This folder is a project-local, provider-neutral playbook. Entries synthesize public design, data visualization, product design, and presentation craft references into operational rules for model prompts. Do not copy source text into generated decks. Use these notes as constraints, heuristics, and QA checks.

Primary public references used while building this library:

- IBM Design Language chart taxonomy: https://www.ibm.com/design/language/data-visualization/charts/
- Observable Plot grammar and examples: https://observablehq.github.io/plot/
- Datawrapper Academy chart guidance: https://www.datawrapper.de/academy
- D3 transition documentation: https://d3js.org/d3-transition
- Duarte presentation structure and sparkline concepts: https://www.duarte.com/blog/business-communication-demands-3-act-story-structure/
- Apple Human Interface Guidelines motion: https://developer.apple.com/design/human-interface-guidelines/motion
- Material Design duration and easing: https://m1.material.io/motion/duration-easing.html
- Primer layout and product design foundations: https://primer.style/product/getting-started/foundations/layout/
- Stripe app design guidance: https://docs.stripe.com/stripe-apps/design
- Bauhaus Archive typography notes: https://www.bauhaus.de/en/research/publications/bauhaus-typography/
- MoMA De Stijl notes: https://www.moma.org/calendar/events/270
- Cooper Hewitt notes on Armin Hofmann and Swiss Style: https://www.cooperhewitt.org/2018/08/05/aharmonyofcontrasts/
- beautiful-html-templates by zarazhangrui / zarazhangrui: https://github.com/zarazhangrui/beautiful-html-templates
  - License: MIT.
  - Used as an adapted template-style reference library. SlideLeaf entries summarize and transform each template's metadata, mood, tone, palette, typography, layout grammar, density, and adaptation constraints into provider-neutral style-direction playbook entries.
  - Runtime/navigation code from the source templates is not imported; SlideLeaf renderer owns navigation, counters, progress, and active slide state.
